1.Una variable es donde se guarda (y se recupera) datos que se utilizan en un programa.
2.Un String (cadena) es una secuencia de caracteres de cero o más caracteres correspondientes al código.Las cadenas se utilizan para almacenar caracteres individuales, palabras y oraciones completas.
3.Una función es una sección de un programa que calcula un valor de manera independiente al resto del programa.
4.Un if en programación se utiliza para evaluar una expresión condicional: si se cumple la condición (es verdadera), ejecutará un bloque de código. Si es falsa, es posible ejecutar otras sentencias.
5.El tipo de dato lógico o booleano es en computación aquel que puede representar valores de lógica binaria, esto es 2 valores, que normalmente representan falso o verdadero.
